So the internet is the same as the web right?

The Internet

Is a massive network of networks

Creates a global network of millions of computers

Information that travels over these networks are called protocols

The Internet is considered hardware - because it is physical cables

The Web

Is the ability to access information using the internet

Again, it is just one way information can shared over the internet

It is built on top of the internet.

The Web uses browsers to access documents (web pages) that are linked together

The Web is considered software because it runs on the internet

What is a text editor?

A text editor is where developers can write code. It can read any language and shows the developer the file structure.

- You dont have to download and use a text editor to edit code and test. - Codepen, Thimble, etc. run in the browser

What are the building blocks of web development?

In web development, development is divided up into two responsibilities: front-end and back-end.

- Many developers specialize in one or the other but some do both - this is called being full-stack.
