What can you do with HTML?

It is used to build the structure of a document.

You build out the skeleton of your page using HTML - this is called "mark-up"

MARK-UP - you are telling the page what types of elements you would like to render

So what are the different types of HTML tags that I can use?

List of HTML tags

While there are many different types of HTML tags, here are the few that we will be focusing on today:

Tag Description
<html>...</html> States that the web page should be written in HTML mark-up
<head>...</head> Has instructions for the browser and meta data for the website
<style>...</style> Where stylesheets (CSS) are declared. Is found in the head.
<title>...</title> Declares the title of the page - this appears on the browser tab
<body>...</body> This is the body of the page - where most of your mark-up will go
<b>...</b> Make and element bold
<i>...</i> Make and element bold
<center>...</center> Centers an element on the page horizontally
<ul>...</ul> Creates an unordered list
<ol>...</ol> Creates a numbered list
<li>...</li> Adds an item to a ordered or numbered list
<br> Forces a line break, dropping to the next line
<p>...</p> Creates a paragraph
<hr> Creates a horizontal rule
<img src='...'> Used to display and image
<a href='...'>...</a> Creates a link

But there is one special element on your page.....

The Document Type element

<!DOCTYPE html>

It appears at the top of the document page

It is not a HTML tag

It is actually an instruction to the web browser about what version of HTML the page is written in.

CSS

Cascading Style Sheets

Allows you to style/display HTML elements on the page

You can add CSS to your HTML elements in three different ways

  • Inline - using the style attribute
  • Internal - by attaching a <style> attribute to the document <head>
  • External - by using an external CSS file
When you create a style for an element using CSS, you are creating a CSS rule:
  • selector: this is the "name" applied to HTML elements that identifies which styles will be applied
    p{}
  • property: what you are changing about the element
    .example{}
  • value: the values of the property set - discrete values such as pixles or colors
    #example{}

What are the different ways you can target an element?

Types of selectors:

  • element: applies styles to all the declared elements on the page
  • class: matches all elements with the declared class attribute - can be declared on multipls elements
  • id: matches all elements with the declared id attribute - can only be declared once on one element
